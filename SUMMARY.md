
### 关键算法

#### 1. 文件分割算法
- 基于字符数分割
- 智能寻找段落边界
- 保留重叠区域确保连贯性

#### 2. 合并算法
- 移除重复的重叠部分
- 保持章节完整性
- 无缝拼接美化后的内容

#### 3. 美化策略
- 上下文感知的美化
- 保持原意的结构优化
- 适度的视觉增强

## 🚀 使用流程

### 标准工作流
```
1. 安装依赖 → pip install -r requirements.txt
2. 配置密钥 → 编辑 .env 文件
3. 运行美化 → python main.py document.md
4. 查看结果 → 打开 beautified/ 目录
```

### 高级工作流
```
1. 自定义配置 → 编辑 config.yaml
2. 批量处理 → python main.py ./docs
3. 使用不同风格 → --style colorful
4. 检查日志 → 查看 beautify.log
```

- 不是简单的切割，而是理解文档结构
