# 🎉 MD BeautifyArts v2.1 新功能说明

## ✨ 新增功能总览

### 1. 📄 页面显示美化后的内容

**功能描述：**
- 美化完成后，可以在页面上直接查看美化后的内容
- 支持原始内容和美化后内容切换预览
- 美化后的内容以富文本格式展示（支持 Markdown 渲染）

**使用方法：**
1. 上传或输入 Markdown 文件
2. 点击"开始美化"
3. 在结果区域点击"美化后"标签
4. 查看渲染后的美化效果

**特点：**
- ✅ 实时渲染 Markdown 为 HTML
- ✅ 保留原始格式和美化格式
- ✅ 支持代码高亮显示
- ✅ 一键复制美化内容

---

### 2. ⚙️ 页面配置 API KEY

**功能描述：**
- 无需编辑 `.env` 文件，直接在网页上配置各平台的 API KEY
- 支持所有主流 AI 平台的密钥配置
- 配置保存在当前会话中

**支持的配置项：**
- **OpenAI**: API Key
- **通义千问**: API Key
- **文心一言**: Access Key + Secret Key
- **讯飞星火**: API Key
- **智谱 AI**: API Key
- **腾讯混元**: API Key

**使用方法：**
1. 点击右上角的"⚙️ API 配置"按钮
2. 在弹出的配置框中填写对应的 API Key
3. 点击"💾 保存配置"
4. 系统会自动重新加载可用模型列表

**注意事项：**
- ⚠️ 配置仅在当前会话有效
- ⚠️ 刷新页面后需要重新配置
- 💡 建议同时配置多个模型的密钥以便切换使用

---

### 3. 💬 AI 聊天窗口

**功能描述：**
- 右下角悬浮聊天窗口
- 支持选择不同的大模型进行对话
- 实时切换模型，体验不同 AI 的回答风格

**功能特点：**
- 💬 多模型支持（可选择任意已配置的模型）
- 🎨 美观的聊天气泡界面
- ⚡ 实时响应，流式对话体验
- 📝 支持 Shift+Enter 换行
- 🔄 自动滚动到最新消息
- 🤖 思考中状态提示

**使用方法：**
1. 点击右下角的"💬 聊天"按钮
2. 在聊天窗口顶部选择要使用的模型
3. 输入消息并发送
4. 等待 AI 回复
5. 可以持续对话

**快捷键：**
- `Enter`: 发送消息
- `Shift + Enter`: 换行

**聊天模式：**
- **自由对话**: 日常问答
- **专业咨询**: 技术问题、知识查询
- **创意写作**: 故事创作、文案撰写
- **代码辅助**: 编程问题、代码审查

## 🔧 技术实现

### 后端 API

#### 1. 配置更新接口
```python
POST /api/config
{
    "openai_key": "sk-...",
    "dashscope_key": "sk-...",
    "qianfan_ak": "...",
    "qianfan_sk": "...",
    ...
}
```

#### 2. 聊天接口
```python
POST /api/chat
{
    "message": "你好",
    "model": "qwen-turbo",
    "system_prompt": "你是一个有帮助的助手"
}
```

#### 3. 美化内容下载
```python
GET /api/download/<filename>
```

### 前端组件

#### 1. 设置模态框
- ID: `settingsModal`
- 包含所有 API Key 输入框
- 提交时调用 `/api/config`

#### 2. 聊天窗口
- ID: `chatWindow`
- 可折叠/展开
- 包含消息列表和输入框
- 实时调用 `/api/chat`

#### 3. 内容渲染
- Markdown 转 HTML
- 简单的正则替换
- 支持代码块、标题、列表等

## 🐛 已知限制

### 1. API KEY 配置
- ❌ 不支持永久保存（刷新失效）
- ✅ 解决方案：使用 `.env` 文件永久配置

### 2. 内容渲染
- ⚠️ 仅支持基础 Markdown 语法
- ⚠️ 复杂表格可能渲染不完美
- 💡 建议下载后用专业编辑器查看

### 3. 聊天功能
- ⚠️ 不支持上下文记忆
- ⚠️ 每次对话独立
- 💡 适合快速问答，不适合长对话
