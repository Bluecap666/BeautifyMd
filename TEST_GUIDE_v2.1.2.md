## 🔍 故障排查

### 如果聊天还是说错日期

**检查服务器日志：**
```
应该看到：
收到聊天请求:
  模型：qwen-turbo
  消息：今天几号？
  当前日期：2026 年 03 月 06 日 星期五  ← 这行很重要
```

**如果没有显示当前日期：**
1. 检查 web_app.py 是否正确保存
2. 重启 Flask 服务
3. 清除浏览器缓存

### 如果公众号格式还有问题

**检查浏览器控制台：**
1. 按 F12 打开开发者工具
2. 切换到 Console 标签
3. 查看是否有 JavaScript 错误

**常见错误：**
- `processMarkdownForWechat is not defined` → script.js 未正确加载
- 格式仍然有大量换行 → 需要刷新页面

### 如果美化不完整

**检查 AI 返回的内容：**
1. 点击"美化后"标签
2. 滚动到文档末尾
3. 检查最后一段是否有美化痕迹

**如果没有：**
1. 刷新页面重试
2. 尝试使用不同的模型（如 qwen-plus）
3. 检查服务器日志中的错误信息
