## 🚀 如何使用

### 1. 克隆仓库
```bash
git clone https://github.com/Bluecap666/BeautifyMd.git
cd BeautifyMd
```

### 2. 安装依赖
```bash
pip install -r requirements.txt
```

### 3. 配置 API Key
```bash
# 复制环境变量示例
copy .env.example .env

# 编辑 .env 文件，填入你的 API Key
# 至少配置一个模型的 API Key
```

### 4. 启动 Web UI
```bash
python web_app.py
```

访问：**http://localhost:5000**
